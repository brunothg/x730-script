# x730-script

This is the safe shutdown script for x730

> **NOTE**
> This version targets newer Raspberry Pi OS versions supporting `pinctrl` instead of `/sys/class/gpio` or `raspi-gpio`


## Prerequisites

1. OS supporting systemd
2. `/bin/bash` binary
3. `pinctrl` binary on path
